Drug Toxicity Optimization Using Generative Models

Graduation Project
Role: Co-author & Developer Date: January 2026 Location: Istanbul, Turkey

Project Overview

This graduation project focuses on leveraging generative AI models for drug discovery optimization. The primary goal is to transform toxic molecular compounds into safer alternatives while maintaining structural similarity and drug-likeness properties.

Technical Approach

Fine-tuned REINVENT4 (Mol2Mol architecture) on ChEMBL matched molecular pairs to translate toxic SMILES (Simplified Molecular Input Line Entry System) representations into structurally similar, safer candidate molecules.

Key Features

  • Multi-objective reinforcement learning optimization targeting toxicity, drug-likeness, and structural similarity
  • Training on ChEMBL matched molecular pairs dataset
  • Validation using external toxicity predictors (pkCSM, ProTox)
  • Drug-likeness screening and ADMET property validation

Outcome

Successfully produced non-toxic candidate molecules with maintained structural similarity to original compounds, validated through external toxicity predictors and comprehensive drug-likeness screening protocols.

Advanced AI-Driven Spam Detection Across Digital Communications

Role: Lead Developer Date: October - December 2023 Location: Istanbul, Turkey

Project Overview

This project is a sophisticated endeavor targeting the pervasive issue of spam across digital communication platforms such as emails, YouTube comments, and SMS messages. Developed at Istanbul Technical University, this initiative capitalizes on a robust amalgamation of machine learning techniques.

Technical Approach

  • Traditional ML: Naive Bayes, Random Forest, Support Vector Machine
  • Text Processing: Lemmatization and Clustering techniques
  • Deep Learning: BERT (Bidirectional Encoder Representations from Transformers)
  • Multi-platform classification: Email, YouTube comments, SMS

Outcome

Achieved high accuracy spam detection across multiple platforms, with BERT demonstrating exceptional performance in understanding contextual language patterns for spam identification.

Comprehensive World Cup Data Visualization and Management

Role: Developer Date: October - December 2023 Location: Istanbul, Turkey

Project Overview

An extensive and multifaceted web application designed to visualize and manage data from FIFA World Cup tournaments, spanning both men's and women's competitions. It encompasses CRUD operations through a user-friendly interface.

Database & Data

  • Over 1.58 million data points comprehensive database
  • All 22 men's tournaments (1930-2022)
  • All 8 women's tournaments (1991-2019)
  • Detailed records of squads, matches, players, and awards

Technical Stack

  • Database: MySQL
  • Backend: Python, Flask, Node.js
  • Frontend: React.js, CSS
  • Containerization: Docker

Features

Dynamic navigation system for Tournaments, Squads, Players, Matches, Match ScoreBoard, Group Standings, Managers, Teams, and Awards with full CRUD functionality.

Role: Researcher Date: November - December 2022 Location: Istanbul, Turkey
